# T29_s05_no_repetition — Устранение повторений

## Customer-facing инкремент

Бот не повторяет одну и ту же фразу в каждом ответе. Стадия дела упоминается один раз, а не в 5 из 8 сообщений. Шаблон «вам не нужно ничего делать» исчезает.

## Цель

Передавать последние 3 ответа AI в контекст `_r1_lawyer()` и добавить промпт-запрет на дословное повторение уже сказанного.

## Контекст

В dialog2 фраза «Ваше дело находится на стадии реализации имущества без имущества с 2 апреля 2026 года» встречается в 5 из 8 ответов. История диалога уже хранится в `chat_history` (Supabase) и читается через `SupportSupabaseService`. Нужно извлекать из неё последние ответы AI и передавать в промпт.

⚠️ Конфликт с T27, T28: все трогают `support.py` — координировать.

## Scope

- `services/support.py` — метод `answer()`: извлечь `last_answers` и сформировать `already_said_block`
- `services/support.py` — промпт `_r1_lawyer()`: добавить параметр `already_said`, вставить блок и правило против повторений

## Критерии приёмки

1. Стадия дела не повторяется дословно в каждом ответе подряд
2. Шаблон «не нужно ничего делать» убран из ответов
3. `last_answers` корректно формируется из `chat_history`
4. При пустой истории — блок пропускается, ошибок нет
