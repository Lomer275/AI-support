# T12 — История диалога в R1/R2 + сообщение эскалации

## Customer-facing инкремент

ИИ не задаёт повторно вопросы, на которые клиент уже ответил в этом разговоре. При конфликте («верните деньги!») клиент получает человечный ответ вместо безликого «подождите несколько минут».

## Цель

Передать историю последних 10 сообщений агентам R1-юрист и R1-менеджер, чтобы они учитывали контекст беседы с самого начала анализа. Обновить шаблон ответа координатора при `switcher=true`, разделив его на два случая: конфликт и нейтральный запрос.

## Изменения в коде

- `services/support.py`:
  - `_r1_lawyer(client_context, question, history: list[dict])`:
    - Добавить параметр `history`
    - Формировать `history_text` аналогично координатору
    - Добавить `history_text` после `client_context` в system-промпт с инструкцией «не запрашивай повторно то, что клиент уже сообщал»
  - `_r1_manager(client_context, question, r1_lawyer, history: list[dict])`:
    - Аналогично добавить `history` в параметры и промпт
    - Акцент: «учитывай при оценке эмоционального состояния»
  - `answer()`: передавать `history` в вызовы `_r1_lawyer` и `_r1_manager`
  - `_coordinator()`: обновить блок формата JSON:
    - Добавить поле `"escalation_type": "none" | "conflict" | "request"`
    - `conflict` (агрессия, требование возврата) → фиксированный ответ: «Понимаю вашу озабоченность. Я передаю ваш запрос специалисту — он свяжется с вами в ближайшее время.»
    - `request` (просьба связаться, вопрос про оплату) → «Этот вопрос требует участия специалиста — я передала запрос, вам скоро ответят.»

## Как протестировать

1. Авторизоваться и написать: «Я уже сдал все документы на прошлой неделе»
2. На следующий вопрос про документы проверить: ИИ не предлагает снова их собирать
3. Написать агрессивное сообщение: «Верните деньги, бездельники!!!»
4. Проверить: ответ содержит «Понимаю вашу озабоченность», а не «Подождите несколько минут»
5. Написать спокойно: «Хочу поговорить с менеджером»
6. Проверить: ответ содержит «требует участия специалиста»

## Критерии приёмки

1. `_r1_lawyer` и `_r1_manager` принимают параметр `history` и включают его в промпт
2. ИИ не переспрашивает факты, озвученные клиентом в текущем диалоге
3. При агрессии/требовании возврата — ответ с «Понимаю вашу озабоченность...»
4. При нейтральной просьбе связаться — ответ с «требует участия специалиста...»
5. Координатор возвращает JSON с полем `escalation_type`
